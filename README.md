**Httplug adapter for the cakephp network library**


## Install

Via Composer

``` bash
$ composer require php-http/cakephp-adapter
```

## Testing

Start the development server using

``` bash
vendor/bin/http_test_server
```

Then run the test suite using

``` bash
$ composer test
```
